# PawPal+ Agentic Workflow — Implementation Checklist

## AI Features Being Added
- Rule-based AI agent (PawAgent) that autonomously manages pet care schedules
- Full schedule generation for new/empty pets using species + age + breed templates
- Urgency escalation: overdue tasks get priority boosted automatically
- Coverage checking: missing feeding/activity/hygiene tasks are auto-added
- Conflict resolution: clashing time slots are automatically rescheduled
- Species/age rules: senior dogs get vet checks, cats get litter tasks, etc.
- Visible reasoning log: every agent decision is shown step-by-step in the UI
